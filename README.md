# text-analysis-final
This project analyzes how public discourse on immigration shifted between the 2020 and 2024 U.S. presidential debates using YouTube comment data. After collecting and cleaning 1,816 comments with an adapted YouTube API script, I applied VADER sentiment analysis and TFâ€“IDF to compare emotional tone and dominant topics across the two years.
